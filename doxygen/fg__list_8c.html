struct list_node *const  previous,
struct list_node *const  next

static

Creates a new list element on the heap.

It prepares it for insertion into the list between elements pointed to by previous and next. The data of the newly created element will point to the same memory location as data.

Parameters
[in]datadata of newly created element
[in]previousexisting list element the new element will be inserted after
[in]nextexisting list element the new element will be inserted before
Returns
pointer to the newly created list element or NULL on failure
